# Aadi's Intelligence — Blog Knowledge Bundle

Personal blog and CMS built with Next.js 16, React 19, Tailwind CSS v4, and Prisma + PostgreSQL. Features a rich admin interface with TipTap editor, Markdown content pages, S3/MinIO image uploads, and dark/light mode.
