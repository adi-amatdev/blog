# API

All API routes are under `/api/admin/` and are protected by the [Proxy Auth Guard](../architecture/proxy.md).

- [Admin Posts](./admin-posts.md) — CRUD for blog posts
- [Admin Pages](./admin-pages.md) — Read/update static content pages
- [Admin Login](./admin-login.md) — Authentication
- [Admin Upload](./admin-upload.md) — File upload to S3/MinIO
- [Admin Categories](./admin-categories.md) — Delete categories
- [Admin Set Password](./admin-set-password.md) — Change admin password
